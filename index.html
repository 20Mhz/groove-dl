JTR's Grooveshark Downloader
by George Stephanos (who is JTR)

A song downloader in Python

It's a great way to download music, if you don't know what Grooveshark is.

Dependencies

Python v2.6 (http://www.python.org)
wxPython v2.8 (http://www.wxpython.org)
ObjectListView v1.2 (http://objectlistview.sourceforge.net/python/)

Installation

Windows users:
I have provided self-extracting 7-zip executables containing all dependencies. So you just need to:

  • Run the file downloaded
  • Choose a directory, click Extract
  • Read readme.txt
  • Run groove.exe

Simple enough ?

Linux users:
You will have to do this the hard way (since I haven't yet provided any binary rpm/deb packages) which consists of basically:

  • Installing dependencies:
    • Grabbing latest Python (2.x) .
    • Installing latest wxPython.
    • Getting OLV from its site and running: 
      sudo setup.py install
  • Downloading my two scripts from git (gui.py and groove.py)
  • Executing groove.py.

Should be very obvious to a linux user.
